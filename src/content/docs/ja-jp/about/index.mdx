#### 友達リンクの形式：

`src/data/friends.ts` に追加してください：

```typescript
const friend = {
  link: "https://あなたのサイト.com/",
  title: "あなたの名前",
  avatar: "/friends/あなたの名前.avif",
  desc: "適当に何か一言",
  note: "なんか書く",
};
```
